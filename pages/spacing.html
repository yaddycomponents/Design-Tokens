AntD v5 Size Token Formula
AntD v5 calculates size map tokens from two seed values: sizeUnit = 4px and sizeStep = 4. The base size token = sizeUnit × sizeStep = 16px. All other tokens are multiples of sizeUnit. Our primitive scale is identical to AntD steps 1–8, with one extra step (64px) for page-level layouts that AntD doesn't name.
size = sizeUnit (4px) × sizeStep (4) = 16px  ★ base token
